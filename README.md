# Algorithms for GATE CSE

A comprehensive collection of algorithms in Python organized by category.

## Categories

- **Sorting Algorithms** - Various sorting techniques
- **Search Algorithms** - Finding elements in data structures
- **Graph Algorithms** - Traversal, pathfinding, MST, and more
- **Dynamic Programming** - Classic DP problems
- **String Algorithms** - Pattern matching and string manipulation
- **Numerical Algorithms** - Mathematical computations
- **Geometric Algorithms** - Computational geometry
- **And many more...**

Each algorithm is implemented in a separate Python file for easy reference and learning.
